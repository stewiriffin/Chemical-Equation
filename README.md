# Chemical Equation Balancer

A web application for balancing chemical equations with step-by-step explanations.

## Features

- Two-column responsive layout
- Interactive equation input
- Predefined example equations
- Step-by-step balancing explanation
- Dark mode support
- Built with React, TypeScript, and Tailwind CSS

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Supported Equations

The application currently supports balancing the following equations:

1. H2 + O2 → H2O
2. CH4 + O2 → CO2 + H2O
3. Fe + O2 → Fe2O3
4. N2 + H2 → NH3

## Usage

1. Enter a chemical equation in the textarea on the left
2. Click "Balance Equation" or press Enter
3. View the balanced equation and step-by-step explanation on the right
4. Click on any example equation to quickly test the balancer
